# Data Jobs Dashboard

## Overview

The **Data Jobs Dashboard** is a Power BI project designed for **job seekers**, **career transitioners**, and **professionals exploring new opportunities in data**. It was built to solve a common problem: information about the data job market is often scattered across multiple platforms and difficult to interpret.

Using a real-world dataset of **2024 data job postings**, this dashboard brings together key information on **job titles, salaries, locations, and required skills** into a single, interactive report. The goal is to make it easier to explore market trends, compare compensation, and identify the most valuable skills in today’s data job landscape.

## Project Objective

This dashboard helps users answer important questions such as:

- Which skills are most in demand for data jobs?
- Which data roles offer the highest salaries?
- How many skills are typically required per job posting?
- What are the median yearly and hourly salaries in the market?
- How do opportunities vary by job title and country?
